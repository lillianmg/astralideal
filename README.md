# AstralIdeal.com

Custom astrology and tarot practice website for [AstralIdeal.com](https://astralideal.com).
---
## Tech Stack

- **HTML, CSS, Vanilla JavaScript**
- **Three.js r128** for animated hero with zodiac rings, constellation particles, nebula starfield, and mouse parallax
- **Netlify** for hosting and form submissions
- **Google Fonts**: Cinzel Decorative, Cinzel, Cormorant Garamond

---

## Pages

| File | Description |
|------|-------------|
| `index.html` | Hero, About Me, Services |
| `contact.html` | Booking form |
| `terms.html` | Terms and Conditions |
| `privacy.html` | Privacy Policy |



## Notes

- No email addresses exposed in source code
- No tracking cookies or analytics
- Copyright year updates automatically via JavaScript
- Contact form submissions delivered to owner email via Netlify notifications
